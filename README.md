# HoodRootsSMP Website

This is the official website for the **HoodRootsSMP Minecraft server**.  
It includes:

- Home/hero section with server welcome and animations
- Server store with ranks and perks ($ prices)
- Join section with server IP and live player count
- Dark/Light theme toggle
- Server history page
- Fully responsive and mobile-friendly design

**Server IP:** HoodRootsSMP.minehut.gg

This site is hosted for free via **GitHub Pages**.

---

## How to Use

1. Clone or download this repository
2. Open `index.html` in your browser to view the site
3. Edit HTML/CSS/JS files to customize content
